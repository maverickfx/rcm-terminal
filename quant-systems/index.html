🖥️ Frontend Architecture Code Stack
Vanilla HTML/CSS/JS
No framework. Intentional — maximises portability across devices, no build step, deploys as static files. Works offline with cached data.
Shared Module: rcm-utils.js
Central RCM object. Two-layer data (IB MCP → FMP REST). Formatters, header/footer builders, source badge, FMP key prompt. All pages import this.
Shared Style: rcm-style.css
Brand token system (navy/gold). All RCM components: header, tabs, panels, KPI tiles, tables, buttons, forms, alerts, spinner, footer, print CSS.
State Persistence
localStorage with rcm_ prefix. Saves: FMP API key, symbol selection, TF/chart type preferences, setup state. No server-side sessions needed.
